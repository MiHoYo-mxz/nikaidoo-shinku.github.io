# 函数概念

  • 函数就是把 功能代码 封装到函数中，调用函数的时候，就会执行函数里面的代码

  • 函数是 一等公民

    • 这意味着，函数可以作为 别的函数的参数 、 函数的返回值 ， 赋值给变量 或 存储在数据结构 中
    • 通常我们对作为头等公民的编程方式，称之为 函数式编程
    • JavaScript 就是 符合函数式编程 的语言，这个也是 JavaScript 的一大特点

  • 函数本身也是一个对象，无论函数是如何创建的，函数都是一个值（这个值的类型是一个对象）

  • 同名函数会被覆盖

# 函数参数

  • 函数参数就是一些在函数内部不确定的值，需要调用的时候才可以确定的值，我们就会做成形参
  • 形参与实参个数差别：
    1. 若实参比形参个数少，没有得到实参的形参输出为 undefined
    2. 若实参与形参个数相同，则保持一致
    3. 若实参比形参个数多，超出形参总个数之外的实参将被忽略
  • 形参 ：
    1. () 小括号里面放的是形参列表，形参不是必须的，函数允许没有形参
    2. 形参就是函数体中用到的些值不是固定的，我们需要调用函数的时候，才可以确定的值，就会通过形参实现
    3. 形参其实就是变量，是一个保存数据的容器
    4. 函数名.length 可以获取形参的总个数
  • 实参 ：
    1. 调用函数是给函数名加上 ()，就可以执行函数里面函数体代码，() 里面可以传递实际参数，简称实参，实参的值会传递给形参，形参就可以用到这个实参的值了，另外实参是可以省略的
    2. 一般而言，定义函数的时候，有几个形参，我们调用函数的时候，就传递几个实参 (一个萝卜一个坑)

# 函数中 arguments 变量

  • 函数中默认会存在 arguments 变量用来接收传递进来的参数
  • 函数中无论是否存在其他变量接收传递进来的参数， arguments 都会对参数进行保存
  • arguments 变量的类型是对象 ( Object )，不是数组类型 ( 伪数组 )，实际上数组类型在 js 中也是一种对象类型
  • 箭头函数无 arguments 对象

# 函数作用域

  • 作用域(scope) 是当前的执行上下文，值和表达式在其中 “可见” 或可被访问
  • ES5 之前没有块级作用域的 (var 定义的变量没有块级作用域)
  • ES5 之前有函数作用域 (scope)
  • 全局 (global) 作用域
    1. 全局作用域下定义变量和函数，在任何地方都可以访问，可以称之为全局变量和全局函数
    2. 全局变量和全局函数会成为 window 对象的属性和方法
  • 函数作用域
    1. 函数内的作用域
    2. 子作用域可以访问父作用域，也就是函数之间可以相互嵌套

# 作用域链

  • 变量取值会逐级往上查，如果当前作用域没有查到值，就会向上级作用域去查找，直到查到全局作用域，这个查找过程形成的链条就叫做作用域链

# 函数的递归实现

  • 在一个函数中调用函数自己本身，达到一定条件时跳出递归循环
  • 先递进，再回归
  • 递归的用法：
    • 第一步：封装一个能解决单一问题的函数 (类似循环体)
    • 第二步：在这个函数中调用当前函数调整问题的规模 (类似循环中的变量更新)
    • 第三步：在函数中设置递归出口如果没有递归出口递归调用无法结束 (类似循环中的判断条件)

# 匿名函数

  • 使用方式

    1. 匿名函数赋值给变量名，实现函数表达式形式

    2. 匿名函数赋值给事件 ( DOM对象事件 )，或者某些对象里面的方法也会使用匿名函数

    3. 立即调用的函数表达式

      • 立即执行函数应用场景

        1. 防止全局变量的命名冲突

        2. 防止 var 定义的变量作用域提升的问题

# 定义函数的方式

  • 通过 function 关键字声明

  • 函数表达式 (匿名函数赋值给变量)

  • 通过 new Function 构造函数创建 ( 少用 )

  • 箭头函数 (ES6)

# return 关键字

  • return 关键字是向函数的调用者返回指定值
  • 如果函数体中没有 return 返回内容，那么函数的调用者将得到 undefined
  • return 关键字只能在函数体中使用
  • 遇到 return 关键字后，会终止函数体代码的向下执行
