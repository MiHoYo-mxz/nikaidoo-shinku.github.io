# 三大家族

  • js 三大家族指的是 offset、scroll 和 client 这三个属性集合，它们都可以用来获取或设置元素的尺寸、位置和滚动等信息
  • offset 家族主要用于获取元素自身的宽度、高度和相对于第一个有定位的父元素的偏移量，包括内容、边框和内边距
  • scroll 家族主要用于获取或设置元素的滚动条位置和滚动区域的宽度、高度，包括内容和内边距
  • client 家族主要用于获取元素可视区域的宽度、高度和相对于浏览器窗口的偏移量，包括内容和内边距

# 三大家族属性

  • Offset 属性包括 offsetWidth、offsetHeight、offsetLeft 和 offsetTop，它们分别表示元素自身的宽度和高度（包括内容、边框和内边距），以及元素距离第一个有定位的父级元素左边和上边的距离（如果没有定位父级，则以 body 为准）。

    1. dom 对象.offsetParent 返回距离当前 dom 对象最近的非静态定位父辈元素对应的 dom 对象，如果父辈元素都是静态定位，将返回 body 元素
    2. dom 对象.offsetHeight 返回 width + 上下 padding + 上下 border
    3. dom 对象.offsetWidth 返回 width + 左右 padding + 左右 border
    4. dom 对象.offsetTop 返回当前 dom 对象距离 offsetParent 元素的上方偏移距离
    5. dom 对象.offsetLeft 返回当前 dom 对象距离 offsetParent 元素的左方偏移距离

  • Scroll 属性包括 scrollWidth、scrollHeight、scrollLeft 和 scrollTop，它们分别表示元素内容的宽度和高度（包括溢出部分），以及元素内容被卷起来的左边和上边的距离。

    1. dom 对象.scrollwidth 得到真实容器的宽度包括 dom 对
      象.clientwidth + 滚动内容的总距离
    2. dom 对 象.scrollHeight 得到真实容器的高度包括 dom 对
      象.clientHeight + 滚动内容的总距离
    3. dom 对象.scrollTop 垂直滚动条滚动的距离
    4. dom 对象.scrollLeft 水平滚动条滚动的距离

  • Client 属性包括 clientWidth、clientHeight、clientLeft 和 clientTop，它们分别表示元素可见部分的宽度和高度（不包括边框），以及元素边框的左边和上边的厚度。

    1. clientWidth 的实际宽度：
      • dom 对象.clientwidth = width+ 左右 padding
    2. clientHeight 的实际高度：
      • dom 对象.clientHeight = height + 上下 padding
    3. clientTop 的实际宽度：
      • dom 对象.clientTop = border.top (上边框的宽度)
    4. clientLeft 的实际宽度：
      • dom 对象.clientLeft = border.left (左边框的宽度)

  • 这些属性只能用于获取不能用于设置，获取到的是不带单位的数值

# 三大家族的应用场景

  • 使用 Offset 属性可以实现拖拽效果、放大镜效果、吸顶效果等。

    • 放大镜效果代码示例：

  • 使用 Scroll 属性可以实现滚动条监听、返回顶部、无限加载等。

    • window.scroll () 滚动窗口至文档中的特定位置。参考文档: https://developer.mozilla.org/zh-CN/docs/web/API/Window/scroll

    • 语法一：

      • window.scroll (x-coord 橫坐标，y-coord 纵坐标)

    • 语法二：

      • window.scroll (options 配置对象)

    • 如果你想使用 scrollTo () 方法，你需要以下几个步骤：

      • 获取目标元素，如按钮或锚点。
      • 给目标元素添加点击事件，当点击时，调用 window.scrollTo () 方法。
      • 传入目标元素的横纵坐标作为参数，或者传入一个包含 left、top 和 behavior 属性的对象作为参数。
      • behavior 属性表示滚动行为，可以是 smooth（平滑过渡）或 instant（瞬间滚动）。

    • 具体的代码示例和解释，请参考以下网址： 1: 使用 scrollTo/scrollTop 让页面元素滚动到指定位置，并设置滚动动画 2: 关于 window.scroll 系列方法汇总 & 滚动到指定元素位置

  • 使用 Client 属性可以实现鼠标跟随、自适应布局、弹出层居中等。

# scroll 事件，给页面绑定滚动事件，获取页面滚动距离

  • 监听浏览器页面滚动事件一般使用 window 对象来监听滚动

    • window.onscroll
    • window.addEventListener("scroll", foo)

  • 获取页面的滚动距离

    • 需要通过 body 对象或者 html 对象获取页面滚动距离

    • 如果存在 <!DocType html>, 使用 html 对象

    • 如果不存在 <!DocType html>, 使用 body 对象

    • 通过 window.pageYoffset 也可以获取页面垂直滚动距离，但是该属性需要 IE9 以后的主流浏览器才支持

    • 通过 window.pageXoffset 也可以获取页面水平滚动距离，但是该属性需要 IE9 以后的主流浏览器才支持

# window.getComputedStyle () 方法

  • getComputedStyle () 方法是用于获取指定元素的 CSS 样式的方法。获取的样式是元素在浏览器中最终渲染效果的样式，包括外部和内部样式表、继承样式和浏览器默认样式。

  • getComputedStyle () 方法返回一个 CSSStyleDeclaration 对象，该对象包含了所有 CSS 属性和值。可以通过调用对象的 API 或者使用 CSS 属性名作为索引来访问单个属性值。

    getComputedStyle () 方法有两个参数：

    • element: 必需，要获取样式的元素。

    • pseudoElement: 可选，伪类元素，如 ":before" 或 ":after"。
