every & some

  • every 和 some 是 JavaScript 数组的两个方法，它们都可以用来对数组中的每个元素执行一个回调函数，并返回一个布尔值。

  • every 方法测试数组中的所有元素是否都满足回调函数的条件，如果是，就返回 true，否则返回 false。

  • some 方法测试数组中是否有至少一个元素满足回调函数的条件，如果有，就返回 true，否则返回 false。

# reduce & reduceRight

  • reduce () 方法是一个数组方法，它接收一个函数作为累加器，数组中的每个值（从左到右）开始缩减，最终计算为一个值。reduce () 方法返回函数的累积结果。reduce () 方法不会执行空数组元素的函数。reduce () 方法不会改变原始数组。

  • reduce () 方法接收两个参数，一个是函数，一个是初始值。函数参数有四个子参数，分别是累加器，当前值，当前索引和数组对象。初始值参数是可选的，用于指定函数的初始值。如果没有指定初始值，数组的第一个元素会被用作初始值，迭代会从第二个元素开始。

  • reduceRight () 方法是 JavaScript 中的一个数组方法，它可以从右到左将数组中的元素通过一个函数累加为一个单一的值。

    • 它的语法是：

      • array.reduceRight(function(total, currentValue, currentIndex, arr), initialValue)
      • 其中，function 是必需的参数，用于执行每个数组元素的操作。total 是初始值或者上一次调用函数的返回值，currentValue 是当前元素的值，currentIndex 是当前元素的索引，arr 是当前元素所属的数组对象。initialValue 是可选的参数，用于指定传递给函数的初始值。

  • find () 方法是一个数组方法，它返回满足提供的测试函数的数组中的第一个元素的值。如果没有值满足测试函数，它返回 undefined。

# includes

  • includes 方法是 JavaScript 中的一个字符串方法，它用于判断一个字符串是否包含另一个指定的字符串。如果包含，它返回 true，否则返回 false。
  • includes 方法的语法是：
    • string.includes(searchvalue, start)
    • 其中 searchvalue 是要搜索的字符串，start 是可选的参数，表示从哪个位置开始搜索，如果省略或小于 0，则默认从 0 开始。
