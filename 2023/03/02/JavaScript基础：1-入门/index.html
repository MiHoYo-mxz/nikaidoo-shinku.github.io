# js 三个常用输出语句

  • console.log ("内容", "内容"...) 在控制台输出内容，不解析 html 元素，控制台中输出以空格隔开

  • document.write ("内容") 把内容输出到 body 标签中，也就是把内容作为 body 标签的标签内容，解析 html 元素

  • window.alert ("内容") 弹窗的形式显示内容

    • 注意：window 对象的属性或者方法都可以省略 window

# js 变量

  • JavaScript 是一种弱类型脚本语言，所谓弱类型指的是定义变量时，不需要什么类型，在程序运行过程中会自动判断类型。

  • 变量声明但未付值时，输出的值为 undefined

  • 变量未声明输出会直接报错

  • 严格区分大小写

# js 数据类型

  • ECMAScript 中定义了 6 种原始类型：

    • Boolean
    • String
    • Number
    • Null
    • Undefined
    • Symbol （ES6 新定义）

  • 基本数据类型：只能存储一个值

    • number（数值）
    • boolen（布尔）
    • null（空）
    • undefined（未定义）
    • string（字符）
    • symbol（es6 新定义，表示独一无二的值）

  • 复合数据类型：至少存储一个值，可以存储多个值

    • object（对象）
    • array（数组）

# typeof

typeof xxx 得到的值有以下几种类型： undefined boolean number string object function 、 symbol ，比较简单，不再一一演示了。这里需要注意的有三点：

  • typeof null 结果是 object ，实际这是 typeof 的一个 bug，null 是原始值，非引用类型
  • typeof [1, 2] 结果是 object ，结果中没有 array 这一项，引用类型除了 function 其他的全部都是 object
  • typeof Symbol() 用 typeof 获取 symbol 类型的值得到的是 symbol ，这是 ES6 新增的知识点

# 循环结构

  • 循环四要素

    1. 初始化变量（表示循环开始的初始值值是什么）
    2. 循环条件（什么时候循环开始，什么时候循环结束）
    3. 变量更新（改变变量的值，如果不改变，容易出现死循环）
    4. 循环体（就是重复执行的代码）

  • while 循环

  • do...while 循环

  • for 循环语法

  • 循环结构关键字

    • break（跳出整个循环）
    • continue（跳出本次循环）

# window.prompt ("标题", "内容")

  • 点击取消得到的是 null 值，点击确定得到的是 string 类型的值

# 数组

  • 数组是一组数据有序的集合，属于复合数据类型，至少可以存储一个值

  • 通过 索引 输出数组中不存在的元素时会显示 undefined

  • 数组元素可以是任意数据类型

  • 数组变量名.length 可以获取数组长度，数组长度是动态值，根据数组元素的个数而变化

  • array.push()

    • push() 方法可以向数组尾部添加单个或多个数组元素

  • array.length = 0

    • 可以使用数组长度清空数组，设置数组长度为 0

  • array[array.length - 1]

    • 找到数组最大下标

  • 定义数组

  • 数组遍历方法
