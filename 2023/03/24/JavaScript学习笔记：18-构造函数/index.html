# 构造函数

  • 构造函数是专门用来生成实例对象的函数。它就是对象的模板，描述实例对象的基本结构。在 JavaScript 中，任何函数都可以作为构造函数，只要使用 new 关键字来调用它。

  • 构造函数的优点是可以区分对象的类型，并且可以使用原型对象来共享属性和方法。构造函数的缺点是需要使用 new 关键字来创建对象，而且每个实例都有自己的属性和方法。

# 构造函数工作原理

  • 构造函数是一种用来创建和初始化对象的特殊函数。在 JavaScript 中，任何函数都可以作为构造函数，只要使用 new 关键字来调用它。

  • 当一个函数被用作构造函数时，它会执行以下步骤：

    • 创建一个空的简单 JavaScript 对象（即 {}）；
    • 为新创建的对象添加__proto__属性，将该属性链接至构造函数的原型对象；
    • 将新创建的对象作为 this 上下文；
    • 执行函数体内的代码（为新创建的对象添加属性）；
    • 返回新创建的对象，除非构造函数返回了一个非空对象。

# 构造函数的优缺点

  • 构造函数有以下一些优点：

    • 它可以创建相同类型的多个对象，避免重复代码。
    • 它可以使用 instanceof 和 constructor 属性来判断对象的类型和来源。
    • 它可以使用 prototype 属性来给所有对象实例添加共享的属性和方法，提高性能和内存利用率。

  • 构造函数也有以下一些缺点：

    • 如果在构造函数内部定义方法，那么每个对象实例都会创建一个新的方法，占用更多的内存空间。

    • 如果在全局作用域中定义相同的局部变量，容易造成全局污染，因为 this 指向的可能不是预期的对象。

    • 如果忘记使用 new 关键词来调用构造函数，那么 this 会指向全局对象 window ，导致意外的结果。

    • 如果想要继承其他构造函数或类的属性和方法，需要使用复杂的原型链或者 Object.create () 方法。
