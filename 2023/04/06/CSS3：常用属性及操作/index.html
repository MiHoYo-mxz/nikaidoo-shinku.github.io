# CSS3 特性

  • CSS3 是 CSS3 的升级版，CSS3 新增一些特性 (CSS 属性), 比如变形，过渡，圆角边框，图片边框等

  • CSS3 大部分属性浏览器都支持了，少部分属性需要设置特定浏览器私有前缀

    • 常用的前缀有：

      1. -moz 代表 firefox 浏览器私有属性
      2. -ms 代表 IE 浏览器私有属性
      3. -webkit 代表 chrome、safari 私有属性
      4. -o 代表 opera 私有属性

    • 通过 canIUse 网站，可以查看指定属性对应在不同浏览器的兼容性 网址: https://caniuse.com

# 盒子阴影

  • box-shadow: 水平偏移量 垂直偏移量 模糊程度 阴影大小 阴影颜色；
    • 水平偏移量 垂直偏移这两个参数是必选，其他参数都是可选的
    • 水平偏移量 垂直偏移量 阴影大小这三个参数取值可以写 正负 值
    • 模糊程度只能 正 值
    • box-shadow: 10px 10px 5px 10px green;
  • 默认是外阴影，如果需要内阴影 则添加一个参数 inset
    • box-shadow: 10px 10px 5px 10px green inset;
  • 多重阴影 每组阴影之间使用逗号隔开

# 文字阴影

  • text-shadow: 水平偏移量 垂直偏移量 模糊程度 阴影颜色；
    • text-shadow: 5px 5px 3px red;
  • 多重阴影 每组阴影之间使用 "逗号" 隔开

# 过渡

  • CSS3 的过渡可以让 CSS 属性变化是有过程的 需要从一个具体值到另一个具体的值

  • transition-property 指定应用过渡属性的名称，默认值 all (所有可被动画的属性都表现出过渡动画) 取值 none (没有属性过渡动画) 取值属性名称

  • transition-duration 属性以秒或毫秒为单位指定过渡动画所需的时间。默认值为 0s，表示不出现过渡动画。

  • transition-delay 属性规定了在过渡效果开始作用之前需要等待的时间。 值以秒（s）或毫秒（ms）为单位，表明动画过渡效果将在何时开始。取值为正时会延迟一段时间来响应过渡效果；取值为负时会导致过渡立即开始。 默认值为 0s

  • transition-timing-function CSS 属性受到 transition effect 的影响，会产生不断变化的中间值，而 CSS transition-timing-function 属性用来描述这个中间值是怎样计算的。实质上，通过这个函数会建立一条加速度曲线，因此在整个 transition 变化过程中，变化速度可以不断改变。 过渡曲线 默认取值 ease (规定慢速开始，然后变快，然后慢速结束的过渡效果) 常用取值 linear (规定从开始到结束都保持相同速度的过渡效果)

  • ❤transition CSS 属性是 transition-property、transition-duration、transition-timing-function 和 transition-delay 的一个简写属性

  • ❤并不是所有的 CSS 属性都可以进行过渡效果：例如 display 加过渡效果无效

# 过渡问题

  • 当指定了元素需要过渡的属性后，默认值 all (所有可被动画的属性都表现出过渡动画)，只要元素的该属性变化就会发生过渡，即使是在 js 中操作元素属性变化

# transitionend 事件

  • transitionend 事件的事件处理函数，在某个 CSS transition 完成时触发。

# 2D 变形 (CSS3) transform

  • 二维坐标系分为两个轴，水平的 x 轴，垂直的 y 轴
    • 水平向右为正，垂直向下也为正
  • CSS transform 属性允许你旋转，缩放，倾斜或平移给定元素。这是通过修改 CSS 视觉格式化模型的坐标空间来实现的。
  • 平移：
    • transform: translateX(-30px);
    • transform: translateY(82px);
    • transform: translate(230px, 50px);
    • 平移取值还可以写百分比 相对自身的宽度或者高度：
      • transform: translateX(-30%);
      • transform: translateY(82%);
      • transform: translate(-50%, -50%);
  • 旋转：
    • transform:rotate (角度值) 数值带 deg 单位 360deg 简单理解就是 360 度就是一圈 数值带 turn 单位 1turn 等于一圈
    • 默认是沿着元素的中心点旋转
      • transform: rotate(60deg);
      • transform: rotate(0.25turn);
  • 缩放：
    • transform: scaleX(-1.2);
    • transform: scaleX(0);
    • transform: scaleX(1);
    • transform: scaleX(1.5);
    • transform: scaleY(1.8)
    • transform: scale(1.5);
  • 倾斜：
    • transform: skewX(30deg);
    • transform: skewX(-60deg);
    • transform: skewY(0.1turn);
    • transform: skew(30deg);

# transform-origin 属性可以调整元素转换变形的原点

  • transform-origin CSS 属性让你更改一个元素变形的原点。

  • 转换起点是应用转换的点。例如，rotate () 函数的转换原点是旋转中心。（这个属性的应用原理是先用这个属性的赋值转换该元素，进行变形，然后再用这个属性的值把元素转换回去）

  • 默认的转换原点是 center

# 一个元素同时使用多个变形

  • 可以同时使用多个变形 使用空格隔开
  • 注意：旋转以后，会改变坐标系
    • transform: rotate(60deg) translateY(200px);
    • transform: translateY(200px) rotate(60deg) scale(1.5);

# CSS3 动画

  • 使用 @keyframes 定义动画

  • 关键帧 @keyframes at-rule 规则通过在动画序列中定义关键帧（或 waypoints）的样式来控制 CSS 动画序列中的中间步骤。和 转换 transition 相比，关键帧 keyframes 可以控制动画序列的中间步骤。

  • 使用 animation 属性或者其子属性，配置动画

  • animation-name 指定由 @keyframes 描述的关键帧名称。

  • animation-duration 设置动画一个周期的时长。 默认值 0s

  • animation-delay 设置延时，即从元素加载完成之后到动画序列开始执行的这段时间。

  • animation-timing-function 设置动画速度，即通过建立加速度曲线，设置动画在关键帧之间是如何变化。

  • animation-iteration-count 设置动画重复次数，可以指定 infinite 无限次重复动画

  • animation-fill-mode 指定动画执行前后如何为目标元素应用样式。 取值 forwards 保留由执行期间遇到的最后一个关键帧计算值 backwards 动画将在应用于目标时立即应用第一个关键帧中定义的值

  • animation-direction 设置动画在每次运行完后是反向运行还是重新回到开始位置重复运行。 常用取值:alternate 动画在每个循环中正反交替播放，第一次迭代是正向播放。确定循环是奇数还是偶数的计数从 1 开始

  • animation-play-state 允许暂停和恢复动画。取值 running 当前动画正在运行，paused 当前动画已被停止

  • 多个动画一起使用

# 动画速度曲线

  • animation-timing-function: 规定动画的速度曲线，默认是 ease

  • linear 匀速

  • steps (n) 指定间隔数量

  • animationend（动画结束）事件在 CSS 动画完成时触发。如果在动画完成前中止了动画，例如将元素从 DOM 中移除，或将动画从元素上移除，animationend 事件不会触发。

# 3D 变形 (CSS3)

  • x 轴 水平向右

  • y 轴 垂直向下

  • z 轴 垂向屏幕，靠近我们的方向的是正值；远离我们的方向是负值

  • 如果想要我们的平面空间可以看到 3d 效果，我们需要给父元素或者父辈元素，一般是父元素，设置一个 perspective 属性，perspective 指定了观察者与 z=0 平面的距离，使具有三维位置变换的元素产生透视效果。z>0 的三维元素比正常大，而 z<0 时则比正常小，大小程度由该属性的值决定。 取值 length 带 css 单位 近大远小

# backface-visibility 属性

  • CSS 属性 backface-visibility 指定当元素背面朝向观察者时是否可见。

  • 取值: visible 默认值 背面朝向用户时可见。

  • 取值: hidden 背面朝向用户时不可见。

# 3D 呈现 transform-style

  • CSS 属性 transform-style 设置元素的子元素是位于 3D 空间中还是平面中。

  • 取值：

    • flat 设置元素的子元素位于该元素的平面中。

    • preserve-3d 指示元素的子元素应位于 3D 空间中。

# 渐变

  • 渐变主要分为线性渐变和径向渐变，另外线性渐变和径向渐变都可以设置为重复渐变

    • background-image: linear-gradient (渐变方向，color-stop1, color-stop2, ...);

    • 如果渐变方向不设置，默认渐变方向是从上到下

# CSS3 - 边框图片

  • border-image-source 用于声明元素的边框图片（border-image）的资源，设置边框图片所在路径

  • border-image-slice CSS 属性将使用 border-image-source 引用的图像划分为多个区域。这些区域组成了一个元素的边框图像。

  • 注意：设置边框图片之前，建议先设置边框，边框图片只是替换边框颜色

  • 默认只设置 border-image-source 属性以后，只有 4 个角有图片

  • border-image-slice 取值是数值，不带单位 或者 百分比 还可以添加一个 fill 保留图像的中心区域并将其作为背景图像显示出来，但其会堆叠在 background 之上。

  • border-image-width 指定了 边界图像 (border image) 的宽度

    • 注意：如果本属性值大于元素的 border-width，边界图像将会向 padding 边缘延展
    • 注意：如果没有设置 border-image-width, 那么将使用 border-width
    • 如果设置了 border-image-width, 将覆盖 border-width 的值

  • border-image-outset 属性定义边框图像可超出边框盒的大小。 取值默认为 0 取值数值带单位，不支持负数 或者 百分比

  • border-image-repeat 定义图片如何填充边框。或为单个值，设置所有的边框；或为两个值，分别设置水平与垂直的边框。取值，stretch 默认值拉伸 repeat 平铺 round 平铺 当不能整数次平铺时，根据情况放大或缩小图像

  • border-image 简写语法如下；

    • border-image: url(./images/border-stars.png);

    • border-image: url(./images/border-stars.png) 30;

    • border-image: url(./images/border-stars.png) 30 fill;

    • border-image: url(./images/border-stars.png) 30 fill repeat;

    • border-image: url(./images/border-stars.png) 30 fill round;

# CSS3 多列布局

  • CSS3 多列布局，其实就类似报纸排版

  • CSS3 中新出现的多列布局 (multi-column) 是传统 HTML 网页中块状布局模式的有力扩充。这种新语法能够让 WEB 开发人员轻松的让文本呈现多列显示。我们知道，当一行文字太长时，读者读起来就比较费劲，有可能读错行或读串行；人们的视点从文本的一端移到另一端、然后换到下一行的行首，如果眼球移动浮动过大，他们的注意力就会减退，容易读不下去。所以，为了最大效率的使用大屏幕显示器，页面设计中需要限制文本的宽度，让文本按多列呈现，就像报纸上的新闻排版一样

  • 介绍： CSS 新增简单的多列布局，可以将文本内容设计成像报纸一样的多列效果。

    • 参考文档: https://www.runoob.com/css3/css3-multiple-columns.html

  • column-width 设置每列的宽度 取值 auto 默认值 或者 设置一个指定的长度

  • column-count 分成多少列 取值 auto 默认值 或者 数值

  • columns: column-width column-count; 同时可以设置 column-width 和 column-count 的简写属性

    • 如果同时设置了 column-width 与 column-count, 那么将取大优先 用父容器的总宽度除以 column-count 的取值得到一个宽度 这个宽度跟 column-width 比较 谁大就使用谁的宽度

  • column-rule 所有 column-rule-* 属性的简写

    • column-rule-width 指定两列间边框的厚度
    • column-rule-style 指定两列间边框的样式
    • column-rule-color 指定两列间边框的颜色

  • column-gap 指定列与列之间的间隙

  • column-fill

    • 概念：属性控制如何将内容划分为列。内容是平衡的，这意味着在所有列的内容将有相同的高度，或者，当使用自动，只占空间的内容需要。可能的值：
      • balance：默认值，会以大约相同的内容量填充各列，但不允许这些列的高度超过 height。您可能会发现，height 随着浏览器水平地均匀分配内容，这些列的长度将变短。
      • auto：会填充每一列，直到到该列 height 填满为止。所以此值不一定会填充所有列，也不一定会均匀填充。
    • 注意：测试 column-fill 属性的效果的时候，需要父容器不设置宽度，父容器设置一个比较高的高度值

  • column-span 指定元素要跨越多少列 默认取值 1 元素应跨越一列 另一个取值是 all 该元素应该跨越所有列
