# this 关键字

  • this 是一个关键字，它指向当前执行上下文中的一个对象。它的值取决于函数被调用的方式，有以下几种情况：

    • 在方法中，this 指向调用该方法的对象。

    • 在单独使用时，this 指向全局对象，在浏览器中就是 window，在严格模式下是 undefined。

    • 在函数中，this 指向全局对象，在严格模式下是 undefined。

    • 在事件处理程序中，this 指向接收事件的元素。

    • 使用 call ()、apply () 或 bind () 方法可以改变 this 的指向，把它绑定到任意对象上。

  • 在箭头函数中，this 与封闭词法环境的 this 保持一致，不会随着调用方式而改变。

# js 中可以改变函数内部 this 指向的三个方法

  • 在 JavaScript 中，有三个方法可以改变函数内部 this 的指向，它们分别是：

    • call () 方法：该方法可以用一个指定的 this 值和若干个指定的参数值来调用一个函数。

    • apply () 方法：该方法与 call () 方法类似，只是参数是以数组的形式传入的。

    • bind () 方法：该方法可以创建一个新的函数，在新函数中 this 的值被绑定为传入的参数。

# new 关键字

1. 首先创建了一个新的空对象
2. 设置原型，将对象的原型设置为函数的 prototype 对象。
3. 让函数的 this 指向这个对象，执行构造函数的代码（为这个新对象添加属性）
4. 判断函数的返回值类型，如果是值类型，返回创建的对象。如果是引用类型，就返回这个引用类型的对象。
